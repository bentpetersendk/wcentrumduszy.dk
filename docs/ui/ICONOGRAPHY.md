# Iconography

## Role

Icons are functional helpers, not decoration. The visual identity should rely on typography, photography, spacing, and the supplied logo mark.

## Style

- outline icons
- rounded caps and joins
- simple geometry
- no filled pictograms except status dots
- no mystical symbol set

## Stroke Width

- Default: 1.75px.
- Small icons under 18px: 1.5px.
- Large feature icons: 2px maximum.

## Sizes

- Inline utility: 16px.
- Button icons: 18px.
- Navigation/drawer: 20px.
- Empty states: 32-40px.
- Admin status: 14-16px.

## Corner Radius

Icon containers may use 8px radius for square controls. Circular containers are allowed only for close buttons, social icons, and media controls.

## Spacing

Icon and text gap: 8px.

Icon-only buttons: 44px minimum hit area.

## Usage

Use icons for:

- menu
- close
- arrows
- search
- language
- calendar
- location
- time
- play/pause
- upload
- edit
- save
- publish
- warning/success

## When Not To Use Icons

Do not use icons:

- as spiritual decoration
- beside every nav item
- to replace clear copy on primary actions
- in large repeated grids just to fill space
- when the meaning is not immediately obvious
- when photography already carries the emotion

## Logo-Derived Motifs

The logo mark may inspire subtle line rhythm or circular geometry. It should not be fragmented into many decorative icons unless approved.
