# Content Model

## Principles

Everything visible on the public website should be editable in the CMS. No page copy, navigation label, CTA text, SEO metadata, image, or FAQ answer should be hardcoded.

All content types should support:

- `id`
- `slug`
- `language`
- `translation_group_id`
- `title`
- `status`
- `published_at`
- `created_at`
- `updated_at`
- `seo_title`
- `seo_description`
- `social_image`
