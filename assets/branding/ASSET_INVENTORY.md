Status: no client-provided source files were present in the repository when Phase 1 began.

This document is the catalogue that should be updated whenever Joanna supplies logos, photographs, portraits, workshop descriptions, typography references, multilingual naming, or previous page drafts. All production design decisions should prefer Joanna-owned material over stock imagery.

## Catalogue Rules

- Every asset must keep its original filename plus a normalized working filename.
- Every asset must record source, owner, usage rights, language, date received, and approved use cases.
- Portraits and client photos must record consent and whether they can be used for social sharing previews.
- Stock imagery is allowed only when no Joanna-owned image exists for that need and the license is documented.
- Logo files should not be redrawn manually unless the original source format is unusable.
